# 실행방법
## TCP
 1. tcp - MsgReceiver 실행(program args = [port_number])
 2. tcp - MsgSender 실행(projram args = [host_name] [port_number])
 3. MsgSender실행한 창에서 text입력하면 MsgReceiver에서 출력됨
## UDP
 udp 디렉토리의 파일을 실행. TCP와 실행방법은 동일함
